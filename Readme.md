# Advent 2024 in Zig

[Advent of Code 2024](https://adventofcode.com/2024) in the [Zig programming language](https://ziglang.org/).

## Installation

Using [mise-en-place](https://mise.jdx.dev/).

```bash
$ mise install
```

## Running a day

```bash
$ zig build day01
```

## Running tests

```bash
$ zig build test
```

## Running docs

```bash
$ zig build docs
$ ./bin/docs
```
